Synchronous Chat Service

This site is based on the project Synchronous Chat Service. This web chat client will connect to an existing IRC server which will be implemented in the second part of the coursework. This web client will allow users to chat in various channels with other users.

LogIn.xhtml: This is the page where the user is directed to when the site is visited.
Signup.xhtml: This page is visited when the user wants to sign up.
Dashboard.xhtml: The user is directed to this page after signing in.
Chat.xhtml: This chat page is the result of clicking on a channel, The user can also click on people to see who all are online.
