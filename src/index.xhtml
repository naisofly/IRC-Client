A Synchronous Chat Service

This site is based on the project Synchronous Chat Service. This web chat client will connect to an existing IRC server which will be implemented in the second part of the coursework. This web client will allow users to chat in various channels with other users.

The external resources used during the completion of this project were Bootstrap UI elements from: bootswatch.com/united/ & bootply.com/6mdOs5FvKU#
All forms in the following links use HTML5 validated input.
LogIn.xhtml: This is the page where the user is directed to when they want to authenticate themselves & login to use the service. They also have the option of joining or creating a new channel on the IRC server.
Signup.xhtml: This page is visited when the user wants to sign up for an account to use the chat service with their chosen nickname, email and password.
Dashboard.xhtml: The user is directed to this page after signing in, where they can view & browse a list of channels to join.
Chat.xhtml: This chat page is the result of joining on a channel where the user can chat with other participants in realtime. The user can also click on the people tab to see a list of all online users.

Project contributions and status can be tracked at the following github repo: https://github.com/naisofly/IRC-Client/graphs/contributors
